Prompts Used: how do i link a csv file to my html website table?


Outputs Received: Linking a CSV file to an HTML table usually means loading the CSV data with JavaScript and inserting it into your table dynamically. HTML alone can’t read files—you need JS.

✅ Method 1: Load CSV automatically (best for your use case)
1. Put your CSV file in your project

Example:

/project
  index.html
  data.csv
2. Create your HTML table
<table id="dataTable" border="1">
  <thead>
    <tr id="tableHeader"></tr>
  </thead>
  <tbody id="tableBody"></tbody>
</table>
3. Add JavaScript to load the CSV
<script>
fetch('data.csv')
  .then(response => response.text())
  .then(data => {
    const rows = data.split('\n').map(row => row.split(','));

    const header = rows[0];
    const body = rows.slice(1);

    // Add headers
    const headerRow = document.getElementById('tableHeader');
    header.forEach(col => {
      const th = document.createElement('th');
      th.textContent = col;
      headerRow.appendChild(th);
    });

    // Add rows
    const tableBody = document.getElementById('tableBody');
    body.forEach(row => {
      const tr = document.createElement('tr');
      row.forEach(cell => {
        const td = document.createElement('td');
        td.textContent = cell;
        tr.appendChild(td);
      });
      tableBody.appendChild(tr);
    });
  });
</script>
